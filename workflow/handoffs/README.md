# Workflow Handoffs

Upstream agents write Markdown handoffs here for Planner to read before creating task-board tickets.

Current handoff patterns:

- `researcher-to-planner.md` or `researcher-to-planner-*.md`
- `site-auditor-to-planner.md` or `site-auditor-to-planner-*.md`

Use timestamped files when preserving multiple runs matters. Use the canonical file name when Planner should read one latest handoff.
